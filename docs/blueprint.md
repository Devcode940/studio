# **App Name**: KenyaWatch

## Core Features:

- Representative Profiles: Display profiles of elected representatives (President to MCA), including photos, contact information, and constituencies.
- Performance Metrics: Show key performance indicators (KPIs) for each representative. The tool should intelligently weigh information, presenting details like votes garnered and participation records.
- Integrity Report: Employ an AI tool to extract and summarize verified saga/scandal information related to each representative from reputable news sources, creating a short integrity report.
- Leaderboard: Provide interactive rankings of representatives based on performance, integrity, and other metrics.
- County GDP Data: Display county GDP data with options for sorting (high to low).
- Census Data: Present Kenyan census data, sortable by different population categories.
- Secure Logins: Secure user login and authentication system with password hashing and protection against common web vulnerabilities.
- Highlights: Highlight key achievements, significant votes, or important statements made by representatives to provide quick insights.
- Enhanced Security: Implement enhanced security measures such as multi-factor authentication and regular security audits to protect user data and maintain platform integrity.

## Style Guidelines:

- Primary color: #2E86AB (Moderate blue) for trustworthiness and stability.
- Background color: #F0F8FF (Light blue) for a clean, calm backdrop.
- Accent color: #A93226 (Strong red) for important alerts and actions.
- Headline font: 'Poppins' (geometric sans-serif) for a modern, precise feel.
- Body font: 'PT Sans' (humanist sans-serif) for readability and warmth.
- Use simple, clear icons to represent data categories and representative attributes.
- Design a clean, card-based layout for representative profiles, with key data points prominently displayed.
